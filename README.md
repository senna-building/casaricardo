# Casa Ricardo — website

Site estático de apresentação de layout e moodboard inicial.

## Visualizar
Abra `index.html` diretamente no navegador ou execute um servidor local:

```bash
python3 -m http.server 8000
```

Depois aceda a `http://localhost:8000`.

## Publicar no GitHub Pages
Envie todo o conteúdo desta pasta para a raiz do repositório e ative o GitHub Pages a partir da branch principal.
